# Security Policy

We take the security of GLASS seriously.

## Reporting a vulnerability

Please do not open a public issue for security vulnerabilities. Report them privately, by either:

- using GitHub's private vulnerability reporting on the affected repository (the **Security** tab, "Report a vulnerability"), or
- contacting the maintainers through [glasspadelapp.com](https://glasspadelapp.com).

We aim to acknowledge a report within 72 hours and will keep you updated as we investigate and resolve the issue. Please give us reasonable time to address a vulnerability before any public disclosure.

## Scope

This policy covers the repositories in the GLASS Padel organisation.
